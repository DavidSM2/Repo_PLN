Part 1: conjunt de textos de "mundocine"¶

En aquest primer conjunt de dades tenim una srie de crtiques de pellcules de cinema, emmagatzemades en format XML (una crtica per arxiu). Hem preparat una funci de tipus generator que processa el directori on estan els arxius de les crtiques i retorna per cada arxiu XML una tupla amb 4 valors:

  • Nom de la pellcula (string)
  • Resum breu de la crtica (string)
  • Text de la crtica (string)
  • Valoraci de la pellcula (int d'1 a 5)
In [ ]:
import os, re
from xml.dom.minidom import parseString

def parse_folder(path):
    """generator that reads the contents of XML files in a folder
    Returns the <body> of the <review> in each XML file.
    XML files encoded as 'latin-1'"""
    for file in sorted([f for f in os.listdir(path) if f.endswith('.xml')],
                        key=lambda x: int(re.match(r'\d+',x).group())):
        with open(os.path.join(path, file), encoding='latin-1') as f:
            doc=parseString(f.read())

            titulo = doc.documentElement.attributes["title"].value

            btxt = ""
            review_bod = doc.getElementsByTagName("body")
            if len(review_bod) > 0:
                for node in review_bod[0].childNodes:
                    if node.nodeType == node.TEXT_NODE:
                        btxt += node.data + " "

            rtxt = ""
            review_summ = doc.getElementsByTagName("summary")
            if len(review_summ) > 0:
                for node in review_summ[0].childNodes:
                    if node.nodeType == node.TEXT_NODE:
                        rtxt += node.data + " "
                        
            rank = int(doc.documentElement.attributes["rank"].value)
            
            yield titulo, rtxt, btxt, rank

Neteja de text¶

Prepararem aquest conjunt de textos per a entrenar un model per a predir la puntuaci de cada crtica a partir del text de la crtica.
Realitzarem el segent processament:

  • Separar el text en tokens
  • Eliminar els tokens de tipus stop-word, signes de puntuaci o espais
  • Convertir les entitats de tipus PER al token persona
  • Lematizar el text

import spacy

nlp = spacy.load("es_core_news_md")

#definimos funcin de normalizado
def normaliza(texto):
    doc = nlp(texto)
    tokens = [t for t in doc if not t.is_punct and not t.is_space and not t.is_stop] #devuelve los tokens que cumplen las condiciones
    palabras = []
    for t in tokens:
        if t.ent_iob_=='B' and t.ent_type_=='PER':
            palabras.append('persona')
        elif t.ent_iob_=='I' and t.ent_type_=='PER':
            continue
        else:
            palabras.append(t.lemma_) #si no es PER aadimos el lema
    salida = ' '.join(palabras) #junta todos los tokens en un string
    
    return salida

Possibles millores sobre el processament¶

Observem els segents problemes:

  • Algunes paraules no se separen correctament perqu en el text original estan unides per signes de puntuaci.
  • La llista de stop-words de spaCy no cont 'a','e','i'.
  • Alguns lemes mantenen les majscules.\

Redefinim la funci de normalitzaci per a corregir aix:

  • Introdum un espai desprs de determinats signes de puntuaci (".", "?") perqu la divisi en tokens siga correcta
  • Filtrem els tokens amb una longitud d'1
  • Passem a minscules el lema de cada token

Completa la funci per a realitzar aquestes correccions:

def normaliza_bis(texto):
    texto = re.sub(r"([?.])", r"\1 ", texto) #aadimos un espacio despus de "." y "?"
    doc = nlp(texto)
    tokens = [t for t in doc if not t.is_punct and not t.is_space and not t.is_stop and len(t) > 1] #filtramos los tokens que nos interesan
    palabras = []
    for t in tokens:
        if t.ent_iob_=='B' and t.ent_type_=='PER':
            palabras.append('persona')
        elif t.ent_iob_=='I' and t.ent_type_=='PER':
            continue
        else:
            palabras.append(t.lemma_.lower()) #aadimos lema en minsculas
    salida = ' '.join(palabras) #junta todos los tokens en un string
    
    return salida

def extraer_adj(texto):
    texto = re.sub(r'([?.])', r'\1 ', texto) #separamos . y ?
    doc = nlp(texto)
    tokens = [t.lemma_ for t in doc if t.pos_=='ADJ']
    
    return ' '.join(tokens)

def generaCritica(criticas):
    """Funcin de tipo generator que devuelve el
    texto normalizado de cada crtica.
    Entrada:
    criticas: objeto 'parse_folder' que itera
    sobre el directio de las crticas
    Salida:
    texto normalizado de cada crtica"""
    for c in criticas:
        yield normaliza_bis(c[2])

from gensim.utils import simple_preprocess
        
class PreprocesaCriticas(object):
    """Pre-procesa el corpus de crticas con la funcin 'simple_preprocess'
    de la librera gensim
    Entrada: directorio de crticas
    Salida: iterador sobre las crticas (como lista de tokens)"""
    def __init__(self, dirname):
        self.dirname = dirname
 
    def __iter__(self):
        for c in parse_folder(self.dirname):
            yield simple_preprocess(c[2], deacc=True, min_len=2)

Defineix una funci de normalitzaci que faa les segents tasques:

  • Eliminar esments i URL mitjanant un patr RegEx
  • Separar el text en tokens convertint-los a minscules, eliminant els que siguen signes de puntuaci, espais o dgits
  • Eliminar els stop-words d'una llista prpia passada com a argument
  • Eliminar els smbols de puntuaci dels tokens (etiquetes, admiracions, etc.)
  • Eliminar els tokens d'una longitud menor de 2
In [ ]:
import string
#lista de stop-words especficos de nuestro corpus (aproximacin)
stop_words = ['los', 'pero', 'por', 'que', 'una']

patron = re.compile('[{}]'.format(re.escape(string.punctuation))) #elimina smbolos de puntuacin

def clean_text(text, stop_words=stop_words):
    """Limpiamos las menciones y URL del texto. Luego convertimos en tokens
    y eliminamos signos de puntuacin.
    Dejamos tokens en minsculas.
    Como salida volvemos a convertir los tokens en cadena de texto"""
    text = re.sub(r'@[^ ]+', r'', text) #elimina menciones y URL
    tokens = nlp(text)
    tokens = [token for token in tokens if not token.is_punct and not token.is_space and not token.is_digit] #filtra tokens (puntuaciones, espacios y dgitos)
    filtered_tokens = [token for token in tokens if not token.is_stop and len(token) >= 2 and token.is_alpha] #limpia tokens (signos de puntuacin, stop-words y longitud<2)
    filtered_text = ' '.join([token.text for token in filtered_tokens]) #juntam,os como string
    
    return filtered_text
